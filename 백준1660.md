## 해결 포인트
**점화식 및 DP**
- 규칙이 있는 숫자들은 일일이 다 구하기보다는 점화식을 만들어보자
- 예전 값을 어떻게든 쓸 수 있을 것 같다면 dp를 생각해보자

## 내 코드
```python
import math
import sys

n = int(sys.stdin.readline())

four = []

tri_sum = 0
idx = 1
while tri_sum <= n:
    tri_sum += idx * (idx+1) // 2
    four.append(tri_sum)
    idx += 1

dp = [math.inf for _ in range(n+1)]
for i in range(1, n+1):
    for num in four:
        if i == num:
            dp[i] = 1
            break
        if num > i:
            break
        dp[i] = min(dp[i], dp[i-num]+1)
print(dp[n])
```
